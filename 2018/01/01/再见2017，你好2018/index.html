再见 2017，你好 2018

Jan 1, 2018 | 随笔 | 阅读

刚刚结束苏州之旅，在诚品书店看见一个 2017 年的讲堂背景板报。主题是《理想的一年》，可以是理想的时间，理想的地方，理想的电影、理想的伴侣等。 2018 又将如何为自己计划理想的一年。

2017 已经过去，如果《理想的一年》用在 2017 ，那么理想的事情最明显的是通过锻炼实现了体重的增加，理想地方就是西子湖畔。2017年最后的几天，朋友圈疯传十八岁，莫名的给人一种时间都去哪儿了的伤感，原来三十岁离得真的不远。

不知不觉转行做程序员已经三年了，年年都会写总结，回想这几年，从一个城市到另一个城市，工作和学习还是挺充实的。以前写博客总是有一阵没一阵的，缺乏足够的毅力去坚持，而在 2017 年中坚持每月写两三篇技术博客或读书随笔，直到现在的总结。曾慷慨激昂计划要在 2017 年学习 Android things 和前端，然而并没有顺利完成，零零星星的学了些前端知识。在自己不知道做什么的时候，努力都学点，让以后想到做什么的时候，可以容易一些。

2017 身边一直有一群那么常联系的朋友同学，还有谈史论今的同事，一个人的生活尽量让自己不那么宅，尽量表达出自己的想法。

你好，2018，我已经准备好了。

明年重要的事情是把前端学好，不会找任何借口去拖延，不喜欢半途而废。还有就是学英语，将学英语的时间增加。工作要努力，博客要继续，但身体是生命的本钱，锻炼还得坚持。

想给 2018 定一个理想的地方：川西，去成都尝尝美食，去四姑娘山瞧瞧，去稻城亚丁走走。
