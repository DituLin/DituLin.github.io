一、变量定义

使用 var 关键字
  • var a , b , c int
  • var s string = “hello world”
  • 可放在函数内，或放在包内（作用域为函数或包内部）
  • 使用 var() 集中定义变量
编译可以自动识别类型
  • var a,b,c = 2,”string”,true
使用 := 定义变量
  • a,b,c := 3,”sdtr”,true

二、内建变量类型

  • bool ,string
  • (u)int(8,16,32,64) ,uintptr
  • byte,rune （没有 char ，只用 rune）
  • float32(64),complex64(128)
  • 原生支持复数类型（i）
强制类型装换

三、常量的定义

  • const filename = “abc.txt” 可以不用大写（大写表示 public）
  • const 数值可作为各种类型使用
