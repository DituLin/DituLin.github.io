1.OkHttp 特点

OkHttp 是一个高效的 Http 客户端，特点如下：

  1. 支持 HTTP/HTTP2/SPDY
  2. socket 自动选择最好的路线，并支持自动重连
  3. 拥有自动维护的 socket 连接池，减少握手次数
  4. 拥有队列线程池，轻松写并发
  5. 拥有 Interceptors 处理请求与响应（如：透明 GZIP 压缩、LOGGING）
  6. 基于 Headers 的缓存策略
